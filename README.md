# 🎓 JNTUK CGPA Analyzer

A comprehensive academic result analyzer built specifically for **JNTUK B.Tech students**.
Track your SGPA, CGPA, backlogs and visualize your academic performance — all in one place!

---

## ✨ Features

- 📊 **Semester wise SGPA** calculation with credits
- 🎓 **CGPA Calculator** using official JNTUK formula
- 📈 **Visual Charts** — Bar, Line & Subject wise graphs
- ⚠️ **Backlog Tracker** — tracks all failed subjects
- ✅ **Pass/Fail Status** for every subject
- 📄 **PDF Export** — professional result report
- 💾 **Auto Save** — data never lost on exit
- 🎨 **Colorful Terminal UI**

---

## 🎯 Grade Scale (JNTUK)

| Grade | Points |
|-------|--------|
| S     | 10     |
| A     | 9      |
| B     | 8      |
| C     | 7      |
| D     | 6      |
| E     | 5      |
| F     | Fail   |

## 🛠️ Tech Stack

- **Python 3**
- **Matplotlib** — charts & graphs
- **FPDF2** — PDF generation
- **Colorama** — colored terminal UI
- **JSON** — data storage

---

## ▶️ How to Run

### 1. Clone the repository
```bash
git clone https://github.com/udaykiran-org/JNTUK-CGPA-Analyzer.git
cd JNTUK-CGPA-Analyzer
```

### 2. Install requirements
```bash
pip install matplotlib fpdf2 colorama
```

### 3. Run the app
```bash
python main.py
```

## 🧮 CGPA Formula Used

**Official JNTUK Formula:**
```
SGPA = Σ(Credits × Grade Points) / Σ(Credits)
Percentage = (CGPA - 0.5) × 10
```
